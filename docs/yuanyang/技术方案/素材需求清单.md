# 河狸云·个性化精装选配系统 — 客户素材需求清单

## 一、项目概述

本系统为C端业主提供在线精装选配体验，业主可在3D全景场景中切换不同材料/家具/设备，实时查看效果并生成订单。需要客户配合提供以下素材。

---

## 二、素材需求

### 方案A：全景照片模式（推荐优先，最快出效果）

#### 1. 基础全景照片

每个户型的每个房间提供 **一张360°全景照片**，作为默认展示底图。

| 要求项 | 规格 |
|--------|------|
| **格式** | 等距柱状投影（Equirectangular），JPEG 或 PNG |
| **分辨率** | 8192×4096 像素（推荐），最低 6144×3072 |
| **宽高比** | 严格 2:1 |
| **来源** | 酷家乐渲染导出 / 3ds Max + V-Ray 渲染 / Insta360相机实拍 |
| **命名规则** | `{户型编号}_{房间名}_默认方案.jpg`，如 `A2_客厅_默认方案.jpg` |

**需要拍摄的房间（以一个标准三房户型为例）：**

- 客厅（含餐厅区域）
- 主卧
- 厨房
- 卫生间（主卫）
- 书房 / 次卧

#### 2. 风格套餐全景照片

每个房间按 **3-5个风格套餐** 各渲染一张全景照片。每张照片中只更换对应套餐的材料/家具，其他结构不变。

**示例：客厅**

| 套餐名 | 地面 | 墙面 | 背景墙 | 沙发 | 文件名 |
|--------|------|------|--------|------|--------|
| 现代简约 | 灰色瓷砖 | 白色乳胶漆 | 岩板 | 灰色布艺 | A2_客厅_现代简约.jpg |
| 轻奢风格 | 爵士白大理石 | 暖灰微水泥 | 大理石+金属 | 奶油白真皮 | A2_客厅_轻奢风格.jpg |
| 北欧自然 | 白橡木地板 | 鼠尾草绿 | 木格栅 | 复古绿丝绒 | A2_客厅_北欧自然.jpg |
| 中式雅韵 | 胡桃木地板 | 白栎木饰面 | 极简石膏板 | 黑胡桃实木 | A2_客厅_中式雅韵.jpg |

**其他房间同理，每个房间 3-5 张全景照片。**

> 💡 提示：如果使用酷家乐，可以在同一户型方案中切换材质后直接导出全景图，效率最高。

### 方案B：分层素材模式（更灵活，中期升级用）

#### 1. 基础全景照片（同方案A）

每个房间一张基础全景照片，但地面和墙面可以用纯色占位。

#### 2. 材质贴图（无缝纹理）

每种可选材料提供一张 **无缝平铺贴图（Seamless Tileable Texture）**。

| 要求项 | 规格 |
|--------|------|
| **格式** | JPEG 或 PNG |
| **分辨率** | 1024×1024 像素（推荐），最低 512×512 |
| **要求** | 四方连续（上下左右可无缝拼接） |
| **来源** | 建材供应商官网 / 酷家乐素材库 / 实物拍摄后处理 |
| **命名规则** | `{品类}_{品牌}_{型号}.jpg`，如 `地砖_诺贝尔_W63012.jpg` |

**需要提供的材质贴图清单（按房间）：**

**客厅：**
- 地面：3-4种（大理石/木地板/瓷砖/水磨石）
- 墙面：3-4种（乳胶漆色板/微水泥/木饰面）
- 电视背景墙：3-4种（岩板/木格栅/大理石+金属/石膏板）

**主卧：**
- 地面：3种（实木地板色系）
- 墙面：3种（乳胶漆/艺术漆）
- 床头背景：3种（皮革/木饰面/艺术涂料）

**厨房：**
- 地砖：3种
- 墙砖：3种（含小白砖）
- 橱柜门板：3种（白色/木纹/深灰）
- 台面：3种（石英石/岩板/不锈钢）

**卫生间：**
- 地砖：3种
- 墙砖：3种
- 浴室柜面板：3种

**书房：**
- 地面：3种
- 墙面：3种
- 书柜面板：3种

#### 3. 家具/设备素材（PNG抠图）

家具类物品提供 **带透明背景的PNG图片**（正面或3/4视角）。

| 要求项 | 规格 |
|--------|------|
| **格式** | PNG（带Alpha透明通道） |
| **分辨率** | 宽度不低于 1200px |
| **背景** | 透明 |
| **来源** | 品牌官网产品图 / 酷家乐导出 / 专业抠图 |
| **命名规则** | `{品类}_{品牌}_{型号}.png`，如 `沙发_HAY_MAGS-L.png` |

**需要提供的家具PNG清单：**
- 沙发：3-4款
- 床：3款
- 书柜：3款
- 书桌：3款
- 浴室柜：3款
- 淋浴套装：3款（可选，也可用示意图）

#### 4. 区域标注文件（可选）

如果需要精确的材质替换区域，请在基础全景照片上标注可替换区域的坐标范围。可以用简单的矩形标注+JSON描述文件：

```json
{
  "room": "客厅",
  "regions": [
    { "name": "地面", "type": "floor", "yRange": [0.62, 1.0] },
    { "name": "背景墙", "type": "tvwall", "xRange": [0.25, 0.75], "yRange": [0.2, 0.62] },
    { "name": "左墙", "type": "wall", "xRange": [0.0, 0.25], "yRange": [0.2, 0.62] }
  ]
}
```

## 四、注意事项

1. **全景照片的拍摄/渲染机位**：每个房间的相机位置应固定在房间中心偏后（靠门方向），高度约1.5m（人眼高度），确保不同套餐的全景图机位完全一致，切换时才不会"跳"。

2. **材质贴图的色准**：请确保贴图颜色与实物样品一致，建议在D65标准光源下拍摄或使用供应商提供的官方数字素材。

3. **命名规范**：所有文件统一使用上述命名规则，按房间分文件夹组织。

4. **酷家乐导出全景图路径**：方案设计完成 → 渲染 → 全景图 → 导出（选择等距柱状投影格式）。
